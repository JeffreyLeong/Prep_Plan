# Software Engineering Prep Plan

This repository contains a 4-week structured learning plan focused on building foundational software engineering skills through small projects, system design exercises, and coding practice.

---

## Goal

The goal of this repo is to develop:
- Basic programming fluency (Python)
- Problem decomposition using pseudocode
- Ability to structure small software systems
- Early system thinking (requirements → implementation)
- Good coding habits (functions, debugging, Git usage)

---

## Structure

Each project is stored as a simple Python file, with supporting requirement documents in Markdown format.


---

## Weekly Breakdown

### Week 1 - Fundamentals
- Menu-based Calculator
- ATM System Requirements
- Task Manager Spec

### Week 2 - Independence
- To-Do List App
- Uber-like system design
- Git branching basics

### Week 3 - Systems Thinking
- Expense Tracker
- Online Store requirements
- Debugging practice

### Week 4 - Simulation
- Mini system project (combined features)
- Full requirements document
- Clean project structure and workflow practice
